Projects

  • Netizenship. A passive OSINT tool for username recon. Tool is written in python with nil dependency of APIs of any website, whatsoever.

  • Jekyll Static Blog.My Personal static blog site created using Jekyll. Retired this when I moved the contents to Hugo. View the old project here.

  • Network Intrusion Detector. Network Intrusion Detector using Machine Learning on the KDD Cup99 dataset. Program classifies incoming connections as good or bad

  • PyPASAD. This is a python implementation of the original work in the paper ‘Truth will out’, Departure-Based Process-Level Detection of Stealthy Attacks on Control Systems.

  • Covid19India. An interactive data analytics on Covid-19 Pandemic in India using GapMinder.

  • WExDA. WExDA is a web based data exploration tool primarily useful for data preperation/ data analysis stage. This automates the EDA via web ui and is built using streamlit.

  • RemindMe. An attempt to get feet wet with Android Development. One unique feature added is the location service. Demo of the app can be found here.

  • Partisan Bias in News. The project was aimed at using various datamining techniques to analyse the partisan bias on Indian News media. Data was generated through webscrapping and insights were drawn from analytics.

  • EmployeeFeedback DApp. Keeping in mind privacy, transparency, fairness and absolute immunity to bias and manipulation, this DApp is in Etherium blockchain using solidarity smart contracts. Demo video can be found here
